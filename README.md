# CodeCademy Find Your Hat - Node.js Mini Game

## Summary
This mini-game is desgined to be played through node.js, primarily in Visual Studio Code. This project was created during the CodeCademy Front-End Engineer Professional Certification.

## Features
+ Ascii Art Style Game

## Instruction
The objective of the game is a random generated map for your player to navigate, to reach your objective, the hat. If you fall through a hole, you lose.

## Controls
W, A, S, D for movement.
